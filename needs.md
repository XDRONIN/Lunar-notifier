To use the manual location search feature, you will need a free API key from OpenCage Geocoding.

1.  Go to [https://opencagedata.com/](https://opencagedata.com/) and sign up for a free account.
2.  After signing up, you will find your API key on your dashboard.
3.  Copy this API key and paste it into the "Enter OpenCage API Key" field in the application.

The application will then be able to search for any city you enter. The NASA APIs will use the provided `DEMO_KEY`, which has some rate limits but is suitable for general use. For more extensive use, you can get your own NASA API key from [https://api.nasa.gov/](https://api.nasa.gov/) and replace the `nasaApiKey` variable in the `index.html` file.
